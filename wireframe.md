# 와이어프레임 — 니하오두(NihaoDoo) 랜딩페이지

> 레퍼런스: 야나두 영어 (https://www.yanadoo.co.kr/)
> 내 주제: **한국 학생을 위한 중국어 학습 플랫폼**
> 변형 원칙: 화면 구성(레이아웃)만 참고. 로고·사진·카피 등 고유 자산은 가져오지 않고 전부 내 것으로 새로 작성.

## 레퍼런스(야나두) 화면 구성 분석 — 위에서 아래로

| 영역 | 야나두 구성 | 톤 |
|---|---|---|
| 헤더 | 로고 + 메뉴(학습법/후기/혜택) + 우측 CTA 버튼 | 고정(sticky) |
| 히어로 | 큰 카피 "영어의 진리는 짧고 쉽게" + 앱/영상 비주얼 | 밝은 배경 |
| 신뢰 바 | "2,700만 뷰" 등 사회적 증거 숫자 | 강조 |
| 학습법 | "아는단어 확장 학습법" 3단계(단어 뱉기→패턴 추출→무한 확장) | 카드 3개 |
| 특징/AI | "쉬운 100단어 AI 영어 선생님" 등 기능 강조 | 이미지+텍스트 |
| 후기 | 리얼 후기, 유명인 도전기 | 카드 |
| CTA/상담 | "학습 상담 신청" 버튼 | 강조 박스 |
| 푸터 | 회사정보·고객센터·약관·SNS | 어두운 배경 |

핵심 카피 패턴: **"야 너두 ○○할 수 있어"**, **"하루 10분"**, **"짧고 쉽게"** → 친근·동기부여형.

## 내 랜딩(니하오두) 섹션 설계 — 위에서 아래로

1. **헤더 (sticky)**
   - 로고 `你 니하오두` + 메뉴(학습법·특징·후기·수강료) + 우측 `무료 체험 신청` 버튼
   - 모바일: 햄버거 → 바로 신청 모달

2. **히어로**
   - 큰 제목: **"야 너두, 중국어 할 수 있어"**
   - 한 줄 소개: "이미 아는 한자로 시작 → 하루 10분이면 충분"
   - 버튼 2개: `3일 무료 체험` / `학습법 살펴보기`
   - 우측 비주얼: 폰 목업 (오늘의 표현 `加油` 카드 + 학습 진행바)

3. **신뢰 통계 바**
   - 누적 수강생 / 만족도 / 하루 학습시간 / 표현 수 (※ 교육용 가상 수치 명시)

4. **학습법 3단계 (기능 ①)**
   - 한자를 아는 한국인은 중국어가 유리하다
   - ① 아는 한자에서 출발 → ② 병음·성조로 소리내기 → ③ 패턴으로 무한 확장

5. **특징 카드 (기능 ②)**
   - AI 발음 코칭 / 하루 10분 습관 / 한자 연상 암기 / HSK 연결

6. **후기**
   - 별점 + 후기 + 수강생 프로필 카드 3개 (※ 교육용 예시 명시)

7. **수강료**
   - 1개월 / 6개월(인기) / 12개월 (※ 교육용 예시 가격, 실제 결제 없음)

8. **문의 / 상담 CTA**
   - 이름·이메일·수준 입력 → 무료 체험 신청 (모달 + 마지막 강조 배너)

9. **푸터**
   - 로고 · 링크 · 회사정보(가상) · **"본 페이지는 교육용 목적으로 제작되었습니다" 고지**

## 색·글꼴 (style.css `:root`에 모아둠 → 한 번에 교체)

- **중국 상징 레드 + 골드** 팔레트
  - `--red #C8102E` (메인) · `--red-deep #8E1218` (헤더/푸터 대비) · `--red-soft #FBE9E7` (연한 배경)
  - `--gold #E0A82E` (포인트·별점·숫자) · `--cream #FFF9F3` (배경) · `--ink #2A1416` (텍스트)
- 글꼴: 제목 `Black Han Sans`, 본문 `Noto Sans KR`, 한자 `Noto Serif SC`

## 반응형
- 데스크톱 2단 그리드 → 모바일(≤900px) 1단, 메뉴는 햄버거로 축소.

## 비고
- 외부 이미지 미사용(CSS·이모지로 구성) → 저작권 안전.
- 모든 브랜드·수치·후기·가격은 가상. 실제 결제/신청 없음 (교육용).
