# 차이나두 (ChinaDoo) — 한국 학생을 위한 중국어 학습 랜딩페이지

부트니스 × Claude 6주차 과제 — **레퍼런스 클론 → 내 것으로 변형 → Vercel 배포**.

- **레퍼런스**: 야나두 영어 (yanadoo.co.kr) 화면 구성 참고
- **내 주제**: 한국 학생 대상 중국어 학습 플랫폼
- **스택**: 순수 `HTML` + `CSS` + `JS` (빌드 도구 없음, 정적 사이트)

## 구조
| 파일 | 설명 |
|---|---|
| `index.html` | 페이지 마크업 (히어로·통계·학습법·특징·후기·수강료·문의·푸터) |
| `style.css` | 스타일. 색·글꼴은 `:root`에 모아둠 → 한 번에 교체 가능 (오션블루 + 선셋 톤) |
| `script.js` | 신청 모달·발음 듣기 등 인터랙션 |
| `wireframe.md` | 레퍼런스 분석 및 섹션 설계 메모 |
| `assets/china-hero.jpg` | 히어로 배경 (상하이 스카이라인, Unsplash 무료 라이선스) |

## 로컬에서 보기
```bash
python -m http.server 8000
# http://localhost:8000
```

## ⚠️ 고지
본 페이지는 **교육용 목적**으로 제작되었습니다. 야나두의 화면 구성을 학습용으로 참고한 클론 연습 결과물이며, 실재하는 서비스가 아닙니다. 모든 브랜드명·수치·후기·가격은 가상이고 실제 결제나 신청은 이뤄지지 않습니다.
